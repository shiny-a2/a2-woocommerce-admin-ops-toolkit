# Maintenance Notes

## Operational Learnings

- Admin tools should not touch payment state unless that is their only purpose.
- Audit notes are useful when a manager has to reconstruct what happened.
- Loading scripts globally in wp-admin creates avoidable conflicts.

## Maintenance Cadence

- Review capability examples after WooCommerce updates.
- Add HPOS notes as the public samples evolve.
- Keep private meta keys out of public documentation.
