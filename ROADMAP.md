# Roadmap

This repository documents public-safe WooCommerce admin operation patterns.

## Near Term

- Add a capability matrix for order operations.
- Add a sample snapshot interface for reversible edits.
- Document screen-level asset loading rules.
- Add a short ADR for keeping payment/gateway logic out of operations tools.

## Later

- Add synthetic modal screenshots.
- Add fixtures for operation service validation.
- Add export/reporting safety notes.

## Not Planned

- Publishing real order metadata.
- Publishing payment or gateway-specific logic.
- Publishing internal staff workflows.
